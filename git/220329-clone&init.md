##TIL

#clone활용
```
1. github에서 repo를 만든다.
2. git clone {주소} 입력해서 연결
3. touch로 파일 만든 뒤 vi로 수정하고 git add하기
4. 앞접시에 올라간 파일을 git commit
5. commit생성 후 git push로 보냄!

*중간중간 git status로 항상 상태 확인하기!
```

#git init활용
```
1. mkdir로 폴더를 만든다
2. git init입력
3. github에서 remote repo 만든 뒤 주소 복사하고 git remote add {이름}{주소}입력
4. touch로 파일 만든 뒤 vi로 수정하고 git add하기
5. 앞접시에 올라간 파일을 git commit
6. git branch -M main으로 branch이름을 github와 같게 변경
7. git push -u mask main으로 repo에 상관관계 부여해주기

```
